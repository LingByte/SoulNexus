Soul Pet 控制面板

配置桌宠项目后点击「启动桌宠」

本地默认 http://127.0.0.1:7072/api

本地模式：选择含 manifest.json + pet.js 的项目目录（无需 jsSourceId）

Pet Studio → 嵌入 → 复制 loader.js 里的 jsSourceId（不是模板 UUID）

全局快捷键，默认 Ctrl+Alt+P（macOS 为 ⌘⌥P）。保存后生效。

全局切换开始/结束语音，默认 Ctrl+Alt+V。需先配置下方语音凭证。

语音对话（WebRTC · 已对接 embed loader）

桌宠加载后会自动注入 pet-voice-bridge，宠物旁出现「聊天」按钮，或通过上方快捷键通话。 需本地 cmd/voice 服务（默认 7080）与智能体 API 凭证。
